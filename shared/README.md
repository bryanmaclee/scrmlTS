# scrml Shared

Shared state types, error types, and utilities used across compiler, editor, and platform.

## Structure
```
shared/
  types/
    error-types.scrml    — common error enums with renders clauses
    user-types.scrml     — User, Session, Permission state types
    file-types.scrml     — File, Directory state types
  utils/
    date-format.scrml    — date formatting helpers
    string-utils.scrml   — string manipulation
    validation.scrml     — common validation functions
```
